# BobForge Frontend

React + Vite frontend for the BobForge AI App Factory.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **React Router DOM** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Axios** - HTTP client for API calls
- **Recharts** - Charting library (if needed)

## Setup

### Prerequisites

- Node.js 16+ and npm

### Installation

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Create environment file:
```bash
cp .env.example .env
```

4. Configure the API base URL in `.env`:
```env
VITE_API_BASE_URL=http://localhost:3000
```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```

## Routes

- `/` - Home page with idea input
- `/blueprints/:id` - Blueprint dashboard for specific blueprint
- `/blueprints/:id/bob-prompt` - Bob build prompt for specific blueprint
- `/artifacts` - Artifact tracker
- `/bob-evidence` - Bob evidence dashboard

### Legacy Routes (for backward compatibility)
- `/dashboard` - Redirects to latest blueprint
- `/bob-prompt` - Redirects to latest blueprint's prompt
- `/evidence` - Bob evidence dashboard

## API Integration

The frontend communicates with the backend API at `VITE_API_BASE_URL`:

- `POST /api/blueprints/generate` - Generate new blueprint
- `GET /api/blueprints` - Get all blueprints
- `GET /api/blueprints/:id` - Get specific blueprint
- `GET /api/blueprints/:id/export` - Export blueprint as Markdown
- `GET /api/artifacts` - Get all artifacts
- `POST /api/artifacts` - Create new artifact

## Troubleshooting

### Port Already in Use
If port 5173 is busy:
```bash
# Kill the process or change port in vite.config.js
```

### API Connection Issues
- Verify backend is running on port 3000
- Check VITE_API_BASE_URL in .env
- Ensure CORS is configured in backend

### Build Errors
```bash
# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install
```
